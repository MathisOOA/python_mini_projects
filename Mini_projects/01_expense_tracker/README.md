# Expense Tracker 

A simple command-line Expense Tracker built with Python to practice object-oriented programming, file organization, and data management.

## Features

* Add a new expense *(COMPLETED)*
* Delete an existing expense *(COMPLETED)*
* View all recorded expenses *(COMPLETED)*
* Calculate the total amount spent *(COMPLETED)*
* Automatically record the date and time of each expense *(COMPLETED)*

## Technologies

* Python 3
* JSON (data storage)
* `datetime` (standard library)
